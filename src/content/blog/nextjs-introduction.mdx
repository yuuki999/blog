# Next.jsの基本と特徴

Next.jsはReactベースのフレームワークで、Webアプリケーションを構築するための多くの機能を提供しています。

## 主な特徴

### 1. ハイブリッドレンダリング

Next.jsでは、以下のレンダリング方式を選択できます：

- **サーバーサイドレンダリング (SSR)**
- **静的サイト生成 (SSG)**
- **インクリメンタル静的再生成 (ISR)**
- **クライアントサイドレンダリング (CSR)**

### 2. ファイルベースのルーティング

```jsx
// pages/blog/[slug].js
export default function BlogPost({ post }) {
  return (
    <article>
      <h1>{post.title}</h1>
      <div>{post.content}</div>
    </article>
  );
}
```

### 3. App Router

App Routerは、React Server Componentsを活用した新しいルーティングパラダイムです。

```jsx
// app/blog/page.jsx
export default function BlogPage() {
  return <h1>ブログ一覧</h1>;
}
```

## まとめ

Next.jsは、モダンなWebアプリケーション開発において、パフォーマンスと開発者体験を両立させるための優れたフレームワークです。
